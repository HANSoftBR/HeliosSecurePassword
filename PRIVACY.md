# 🔒 Política de Privacidade — Helios Secure Password

**Última atualização:** 2026

---

## 1. Informações Coletadas

O aplicativo **Helios Secure Password** não coleta, armazena, compartilha ou transmite dados pessoais dos usuários.

Todas as informações inseridas no aplicativo, como logins, senhas e anotações, são armazenadas **localmente** no dispositivo do usuário e protegidas por criptografia.

Nenhuma informação pessoal é enviada para servidores externos ou terceiros.

---

## 2. Processamento de Dados

Todo o processamento realizado pelo aplicativo ocorre exclusivamente no dispositivo do usuário.

O aplicativo:

- ❌ Não envia dados para servidores externos  
- ❌ Não utiliza serviços de análise (analytics)  
- ❌ Não utiliza rastreamento  
- ❌ Não exibe publicidade  
- ❌ Não coleta informações pessoais  

Todos os dados permanecem sob controle total do usuário.

---

## 3. Armazenamento e Segurança dos Dados

Os dados armazenados no **Helios Secure Password** são protegidos por criptografia avançada.

- 🔐 Dados armazenados localmente em banco seguro  
- 🔐 Proteção por senha mestra  
- 🔐 Informações criptografadas antes de serem salvas  

### Importante

- A senha mestra **não pode ser recuperada**  
- A perda da senha resultará na perda permanente dos dados  
- O desenvolvedor **não possui acesso** às informações armazenadas  

---

## 4. Permissões do Sistema

O aplicativo pode acessar o sistema de arquivos do dispositivo apenas para:

- Salvar dados criptografados localmente  
- Carregar dados previamente armazenados  
- Exportar e importar backups  
- Acessar arquivos selecionados pelo usuário  

Essas permissões são utilizadas exclusivamente para o funcionamento do aplicativo.

---

## 5. Privacidade do Usuário

O uso do aplicativo é totalmente privado e não exige identificação do usuário.

Não é necessário:

- Criar conta  
- Fornecer e-mail  
- Informar telefone  
- Inserir qualquer dado pessoal  

O controle dos dados é inteiramente do usuário.

---

## 6. Uso do Aplicativo

O **Helios Secure Password** é um software de armazenamento seguro de credenciais.

O aplicativo tem como objetivo proteger e organizar informações sensíveis de forma local e segura.

A segurança dos dados depende do uso adequado da senha mestra e da proteção do dispositivo onde o software está instalado.

---

## 7. Isenção de Responsabilidade

O software é fornecido **"NO ESTADO EM QUE SE ENCONTRA" ("AS IS")**, sem garantias de qualquer tipo.

O desenvolvedor não se responsabiliza por:

- Perda de dados devido ao esquecimento da senha mestra  
- Acesso não autorizado ao dispositivo do usuário  
- Uso inadequado do software  
- Falhas causadas pelo ambiente do sistema operacional  

---

## 8. Alterações nesta Política

Esta Política de Privacidade pode ser atualizada a qualquer momento para refletir melhorias no software ou mudanças legais.

A versão mais recente estará sempre disponível no repositório oficial do projeto.
